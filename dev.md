# Development Notes

## Environment Variables (in .env)
- ANTHROPIC_API_KEY - Claude API key for AI features
- NETLIFY_AUTH_TOKEN - Netlify API token for deployments
- RESEND_API_KEY - Resend API key for email outreach
- FROM_EMAIL - Sender email address
- OUTSCRAPER_API_KEY - Outscraper API key for lead finding
- FIRECRAWL_API_KEY - Firecrawl API key for serverless scraping

## Quick Commands
- `npm start` - Start dashboard server (port 3000)
- `npm test` - Run test suite
- `netlify dev` - Run with Netlify dev environment
